Content Manager - Web Client Installation

Configuration

This instance of the Web Client is yet to be configured, to do this:

  1. go to the Web Client folder (e.g. C:\Program Files\Micro Focus\Content Manager\Web Client),
  2. copy the file 'hprmServiceAPI.config.template' to 'hprmServiceAPI.config',
  3. open notepad.exe using the 'Run as administrator' option,
  4. open hprmServiceAPI.config,
  5. enter your database id in the databaseId attribute, and
  6. if you want to alter any of the workgroup server settings do this also.

After you have completed these steps re-open the Web Client.

Example

<setup databaseId="[Your database Id]" />

  
<workgroupserver port="1137"
                 workpath="C:\Content Manager\WebClientWorkpath"
                 name="local"
                 alternatename=""
                 alternateport="0" />
