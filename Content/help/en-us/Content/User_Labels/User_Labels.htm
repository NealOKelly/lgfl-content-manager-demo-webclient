User Labels

User Labels enable users to create unique categories to allow them to manage records specifically for their

own requirements. It assists users to easily find records and Locations using these User Labels.

To access User Labels in the Content Manager Web Client:

On the Navigation panel, click User Labels.

This will display the User Labels panel on which User Labels can be created, as well as listing your available User Labels.

Creating User Labels

  1. On the User Labels panel, click Create User Label

The User Label entry form appears.

  1. Complete the User Label entry form, see User Label fields for details.
  2. Click Save.

User Label fields

  • User Label Name - mandatory field - type in the name of the new User Label.
  • Icon - from the drop-down list, select an icon to represent the User Label.
  • NOTES - type in the notes or description for the new User Label.
