Working with User Labels

Each User Label is displayed with the following:

  • Hierarchy of the User Label
  • User Label Name

To display further information or edit, click on the User Label. The User Label expands to display additional information as well as the task buttons and icons.

Available Tasks

The icons available for the User Label depends on the Hierarchy of the User Label.

Update

To update the User Label's properties (metadata), click Update.

The User Label fields are displayed. To edit the metadata, either type or select the new details and then click Save.

Delete

To delete the User Label, click Delete.

Browsing via a User Label

From the Records panel, records that have been added to User Labels can be browsed for. Also, records can be added to or removed from User Labels using the Add/Remove record task.

  1. On the Records panel, click Browse via User Label.
  2. Navigate through the displayed list of User Labels.
  3. Click the User Label to view its associated records.

Adding or Removing a User Label from records

For details on adding or removing a User Label from an individual record or a number of tagged records, see Add/Remove Options for details.
