Locations

The Locations feature in the Content Manager Web Client allows users to view and update their Favorite Locations, navigate to associated Content Manager items, create new Locations and create Location type reports.

To access the Locations panel in the Content Manager Web Client:

  1. On the Navigation menu, click Location.

This will display the Location panel, showing the user's Favorite Locations.

Updating a Location

To modify the details of a Location:

  1. On the Locations panel, from the displayed list of Locations, select the Location to be updated.
  2. On the selected Location, click Update.

The Location entry from will be displayed.

  1. Modify the details as required. See Creating a Location - Non-Person Type or Creating a Location - Person Type for details.
  2. Click Save.

Adding Locations to Favorites

  1. On the Locations panel, from the displayed list of Locations, select the Location, or tag multiple Locations, that are to be added to your Favorite Locations list.
  1. Click the Add to Favorites star icon.

The selected Location(s) will be added to your Favorite Locations list, which is accessible from the Location panel.

NOTE: If a Location has been added to your Favorite Location list, the Favorite Location star icon will appear blue.

Removing Locations from your Favorites

  1. On the Locations panel, from the displayed list of Locations, select the Location that is to be removed from your Favorite Locations list.
  2. On the selected Location, click the Remove from Favorites star icon.

The selected Location will be removed your Favorite Locations list, which is accessible from the Location panel.

Setting Location Active Date Range

  1. On the Locations panel, from the displayed list of Locations, select the Location, or tag multiple Locations, whose active period is to be set.
  2. Click Set active date range.

The Active From and Active To dialog appears.

  1. Type the required date to the Active From and/or Active To fields.
    • Active From - the start date that the Location will be active from.
    • Active To - the end date that the Location will be active to.
  2. Click Save.

GPS Location

Using Google Maps, or other mapping solutions, users can apply a global positioning system (GPS) coordinate to a single Location or to multiple Locations. To add or modify a GPS location to a single Location:

  1. On the Locations panel, from the displayed list of Locations, click the Location that is to be updated.
  2. Click Details and then click GPS Location

The GPS Location dialog appears.

  1. Click GPS Location

The GPS Location Browser window appears.

  1. Click on the displayed map to add the GPS location marker(s). If a marker is added to an incorrect location, click on the marker to remove it.
    Map options include:
    • Start your search here - type in the location name or address. The map marker will jump to that location.
    • As Polygon - select this option to to connect the map markers on the map with a complex shape to indicate a region.
    • As Line - select this option to connect the map markers on the map with a line to indicate a region.
    • Traffic - select this option to display traffic information on the map.
    • Clear Markers - removes any markers placed on the map.
  1. Click OK.
  2. On the GPS Location dialog, click Save.

To add or modify a GPS Location for multiple Locations:

  1. On the Locations panel, from the displayed list of Locations, select the Locations that are to be updated.
  2. On the Locations panel, click Details and from the displayed drop-down list, click GPS Location.

The GPS Location dialog appears.

  1. Click GPS Location

The GPS Location Browser window appears.

  1. Click on the displayed map to add the GPS location marker(s). If a marker is added to an incorrect location, click on the marker to remove it.
    Map options include:
    • Start your search here - type in the location name or address. The map marker will jump to that location.
    • As Polygon - select this option to to connect the map markers on the map with a complex shape to indicate a region.
    • As Line - select this option to connect the map markers on the map with a line to indicate a region.
    • Traffic - select this option to display traffic information on the map.
    • Clear Markers - removes any markers placed on the map.
  2. Click OK.
  3. On the GPS Location dialog, click Save.

Show Associated Content Manager Items

The Content Manager Web Client enables you to quickly find and view the records, Activities, To Do Items and requests associated to a Location.

  1. On the Locations panel, from the displayed list of Locations, select the Location, or tag multiple Locations, whose associated items you want to view.
  2. On the selected Location, click Show and then click:
    • Contact Records - displays the records where the selected Location is a Contact Location.
    • Assigned Records - displays the records where the selected Location is the Assignee.
    • Created Records - displays the records created by the selected Location.
    • Owned Records - displays the records where the selected Location is the Owner Location.
    • Checked Out Records - displays the records that are checked out by the selected Location.
    • Activities - displays the Workflow Activities associated with the selected Location.
    • To Do Items - displays the selected Location’s To Do Items.
    • Requests For - displays all requests submitted for records that have the selected Location as their Home Location.
    • Requests By - displays all requests submitted by the selected Location.

    The associated items will be displayed in their search results panel.

Deleting a Location

If you have the correct user permissions, you can delete a Location from Content Manager.

  1. On the Locations panel, from the displayed list of Locations, select the Location that is to be deleted.
  2. On the selected Location, click Delete.

The Confirm delete dialog appears.

  1. Click OK.

Adding Notes to a Location

Additional information about the Location can be added to the Notes section of the item. Notes can be added by Updating a Location from the Location entry form; or they can be added directly from the Location panel.

  1. On the Locations panel, from the displayed list of Locations, select the Location that the Note is to be added to and then click Notes.

The Notes section will be displayed.

  1. Type the additional information into the Notes field, click Add Notes.

The typed Notes will be added to the Location.

To view the most recent Note added, click Show Latest and the displayed Notes will jump to the most recent entry.
