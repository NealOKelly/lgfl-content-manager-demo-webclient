Activities and Documents

Activities can have 3 types of documents associated to them:

  • Working Document - this is a document that is created as a part of completing the Activity.
  • Working Document (review) - this is a document that has been created as a part of a previous Activity and you may need to review or edit it to complete the current Activity.
  • Reference Documents - this is a document that you may need to read or view as a part of completing the Activity.
NOTE: Users of Workflow in the Content Manager Web Client can only attach electronic documents to existing Activity document, they are not able to add new documents to an Activity.
Attaching a document to a Working Document

If an Activity has a Working Document placeholder on it, users may need to attach an electronic document to it to as a part of the Activity process. To add a document to a Working Document placeholder:

  1. On the selected Activity, click Document.

The Records associated to the Activity will be displayed in the Search Results panel.

  1. Click placeholder for the Working Document, and then click Attach Electronic Document.
  2. From the displayed list of Document types, select the type of document that will be added to the Working Document placeholder. Select from:
  • Record Super Copy - this will create a new Content Manager Record based on an existing Content Manager Record and attach it to the Working Document placeholder.
  • Local Electronic File - this will allow you to attach an electronic file from your local directory and attach it to the Working Document placeholder.
  • Record - this will allow you to search for an existing Content Manager Record and attach it to the Working Document placeholder.
  1. Complete the required steps to attach the electronic document (see Attaching a Record Super Copy; or Attaching a Local Electronic File; or Attaching a Record for details).
  2. Click Return to Activity documents to return to the list of associated Activity Documents.
  3. Repeat the process for any other Working Documents, if required.
  4. Click Return to Activities to go back to the list Activities, where the Activity can be completed (see Completing an Activityfor details).

Working with a Reference Document or a Working Document (Review)

See Working with Records for details on how to work with Reference and Working Document (Review) documents.

Attaching a Record Super Copy

A Record Super Copy will create a new electronic document that is based on an existing Content Manager Record.  To attach a Super Copy:

  1. From the selected Working Document, click Attach Electronic Document and then click Record Super Copy from the displayed list of Document types.
  2. Click in the Select Record Type field and select the required Record Type. If the Record Type is not listed, type in the name of the Record Type and then select it from the displayed list.
  3. In the Search for field, type the search method and clause, for example CreatedOn:Today or title:Reef
  4. Click Search.

The search results will be displayed in the Search Results panel.

  1. From the displayed list of Records, click the name of the Record to be attached to the Activity document, and then click Select.

The Record Creation form will be displayed.

  1. Complete the Record Entry form (see Record Entry Form Fields for further details).
  2. Click Save.

The new Record will be attached to the Activity Working Document.

Attaching a Local Electronic File

A local electronic file will create a new Content Manager Record and attach it to the Activity Document. To create

  1. From the selected Working Document, click Attach Electronic Document and then click Local Electronic File from the displayed list of Document types.
  2. Click in the Select Record Type field and select the required Record Type. If the Record Type is not listed, type in the name of the Record Type and then select it from the displayed list.

The Record Creation form will be displayed.

  1. Complete the Record Entry form (see Record Entry Form Fields for further details).
  2. Click Attach Electronic Document and navigate to the location of the file, select the file to be attached to the Record and then click Open.
  3. Click Save.

The new Record will be attached to the Activity Working Document.

Attaching a Record

Selecting to attach a Content Manager Record allow a user to search for an existing Record and attach it to the Activity document.  To attach a Content Manager Record:

  1. From the selected Working Document, click Attach Electronic Document and then click Record from the displayed list of Document types.
  2. In the Search for field, type the search method and clause, for example CreatedOn:Today or title:Reef
  3. Click Search.

The search results will be displayed in the Search Results panel.

  1. From the displayed list of Records, click the name of the Record to be attached to the Activity document, and then click Select.

The new Record will be attached to the Activity Working Document.

Detaching an Electronic Document

If there is a need to remove an attached electronic document on an Activity Working Document placeholder, this can be done using the Detach option.

To detach an electronic document:

  1. From the selected Activity Working Document placeholder, click Detach.

The attached electronic document will be removed.
