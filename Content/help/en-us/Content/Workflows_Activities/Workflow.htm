Workflow

The Workflow feature in the Content Manager Web Client allows users to initiate Workflows based on existing Workflow Templates and to action Workflow Activities.

To access the Workflow and Activities in the Content Manager Web Client:

  1. On the Navigation menu, click Work lists and then click Workflow.

This will display the Workflow panel on which Workflows can be initiated, as well as searching for Activities and Workflows.

Initiating a Workflow

  1. On the Workflow panel, click Initiate a Workflow from Template; or

    2. On the Keyboard, press CTRL + ALT + i

  1. Select the required Workflow Template from the displayed list. To move through the list of Workflow Templates, you can press the DOWN or UP arrow keys on the keyboard and press Enter to confirm the selection.
NOTE: If the Template you require is not listed, type the name of the Workflow Template in the search field and then select it from the displayed results.
  1. In the displayed Name field, type in the Name of the new Workflow, click Save.

The new Workflow will be displayed in the Search Results panel.

Working with Workflows

Users can view the associated Activities and Records on a Workflow.

To view the Activities:

  1. Click on the Workflow name.

The Workflow will expand.

  1. Click Activities.

All associated Activities for the selected Workflow will be displayed in the Search Results panel.  To return to the Workflow, click Return to Workflows in the Workflow panel.

To view the Records:

  1. Click on the Workflow name.

The Workflow will expand.

  1. Click Records.

All associated Records for the selected Workflow will be displayed in the Search Results panel.  To return to the Workflow, click Return to Workflows in the Workflow panel.

See Properties Panel for details on how to view the properties of a Workflow.

For details on working with the Activities associated to a Workflow, see Activities.
