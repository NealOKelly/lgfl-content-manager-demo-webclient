Properties Panel

A customizable Properties Panel is available on all Content Manager Web Client object types. This Properties Panel displays the properties of the selected object.

To view the Properties Panel:

  1. Click on a Content Manager object, for example, an Activity.
  2. Click the Hide/Show Properties button (<<) in the right-hand corner of the screen.

The Properties Panel will be displayed.

To close the Properties Panel:

  1. Click the Hide/Show Properties button (>>) on the Properties Panel.

The Properties Panel will close.

Modifying Properties Panel

The fields displayed in the Properties panel can be modified by a user.

To add properties:

  1. On a selected Content Manager object, in the open Properties panel, click Select properties.
  2. On the displayed Select Properties dialog, from the Available Properties or Fields column, click each property that is to be added to the Properties panel and then click Add. 
    The list of Available Properties or Fields can be searched by typing the property name into the Find a property field above the column.
    The list of Available Properties or Fields can be sorted alphabetically, either ascending or descending, by selecting the A-Z or Z-A options.
  3. Click OK to save the changes.

To remove properties:

  1. On a selected Content Manager object, in the open Properties panel, click Select properties.
  2. On the displayed Select Properties dialog, from the Selected Properties or Fields column, click each property that is to be removed from the Properties panel and then click Remove. 
  3. Click OK to save the changes.

The order that the properties are displayed in on the Properties panel can be changed by:

  1. In the Selected Properties or Fields column, move your mouse cursor over the property that is to be moved, a icon will appear.
  2. Click and hold the property field and move the mouse in the direction the property is to be moved to.
  3. Drag the property field to the required location and release the mouse button.
