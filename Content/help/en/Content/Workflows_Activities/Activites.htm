Activities

The available Activities from the Workflow panel, allow a user to quickly search for those Activities that are:

  • Ready to Start - these are the Activities that are assigned to the currently logged in user and are ready to be started.
  • Incomplete - these are the Activities that are assigned to the currently logged in user and are not completed.

To display the Activities in these available searches, click the Search name.  The Activities will be displayed in the Search Results panel.

Working with Activities

To view the details of an Activity, click the Activity name and it will expand to the actions that can be performed on the selected Activity.

Updating an Activity

  1. To modify the details of an Activity, click Update.

The Update Activity dialogue appears.

  1. Modify the details of the Activity as required. See Update Activity Fields for further details.
  2. Click Save.
Update Activity Fields

Name - displays the name of the Activity.  To modify this name, delete the existing name and type in the new Activity Name.

Assigned to - displays the Location who the Activity is currently assigned to. To reassign this Activity, delete the existing Location name and type in the name of the new Location the Activity will be assigned to, select the Location's name from the displayed list.

Supervisor - displays the Location who the Activity is currently supervised by. To change the Supervisor, delete the existing Location name and type in the name of the new Location the Activity will be supervised by, select the Location's name from the displayed list.

Priority - displays the priority of the Activity. To change the priority, select the new priority from the drop-down list.

Estimated Duration - displays the estimated time it will take to complete the Activity. To change this time, delete the existing duration and type in a new number.

Estimated Cost - displays the estimated cost of completing the Activity. To modify this cost, delete the existing cost and type in a new number.

Reassigning an Activity

If you're unable to complete an Activity or need to pass it onto another person, an Activity can be reassigned. To reassign an Activity:

  1. On the selected Activity, click Reassign.

The Reassign dialogue will display.

  1. Type in the name of the Location the Activity is to be reassigned to and select the name from the displayed list.
  2. Click Save.

Completing an Activity

There are 3 steps to completing an Activity in Content Manager Web Client:

  • Attach a document(s) - see Activities and Documents for further details.
  • Select the result(s)
  • Add a Note

If there is a mandatory requirement for a user to complete these steps, it will be shown by a mandatory (!) indicator on the relevant button.  Also, if the user attempts to complete an Activity without finalising the step a warning message will be displayed.

To complete an Activity with no mandatory steps (or if all mandatory steps are done):

  1. On the selected Activity, click Complete.

The Activity will be automatically completed and be moved to the next Assignee's queue.

If the next Activity is not assigned to anyone, when you complete your Activity, the Reassign Activity dialogue appears with a warning that the next Activity does not have an assignee set . Type in the name of the Location who is responsible for the next Activity, click Save.

NOTE: if you do not have the correct user permissions to Reassign an Activity, the Reassign Activity dialogue still appears, you will get an Access denied error when you try to reassign the Activity, click Cancel on the Reassign Activity dialogue to complete the Activity.

Attaching a Document(s)

For details on attaching document(s) to an Activity, see Activities and Documents for details.

Selecting the Result(s)

If an Activity has multiple possible outcomes, a user must select the result of the Activity before it can be completed. To select the Result(s):

  1. On the selected Activity, click Results.

The list of possible outcomes will be displayed.

  1. Select the appropriate Result(s) for the Activity.
  2. To complete the Activity, click Complete.

Adding Notes

You may have a requirement to put in additional information for an Activity. You can do this by adding Notes. To add a Note to an Activity:

  1. On the selected Activity, click Notes.

The Notes section will be displayed.

  1. Type the Notes into the Notes field.
  2. Click Add to Notes

The typed Note will be added to the Activity.

  1. To complete the Activity, click Complete.
NOTE: Multiple Notes can be added to an Activity. Repeat the above process for each Note that is added.
If multiple Notes have been added to an Activity, clicking Show Latest will jump to the most recent entry.

 

To view the most recent Note added, click Show Latest and the displayed Notes will jump to the most recent entry.
