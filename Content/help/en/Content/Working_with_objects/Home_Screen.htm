Home Screen

The Home Screen of the Content Manager Web Client is a dashboard that enables users to access Records and complete actions such as Searching, creating and updating records; creating, viewing and actioning Requests; viewing and completing Workflows; viewing and creating Locations; viewing, creating and updating To Do Items, as well as viewing and updating Check In Styles via the Navigation menu. As well as to be able to quickly explore recently created, used or favourite records.

Home Screen Dashboard

By default, the Home Screen Dashboard also displays records relevant to the currently logged on user. The 3 default dashboard panels are:

  • Recent Documents - lists your recently used electronic record. This list is sorted by the most recent created date.
  • Favourites - lists the records you have added to your list of Favourite records. This list is sorted alphabetically.
  • Documents Checked Out - lists the records you have Checked Out.  This list is sorted by the most recent Checked Out date.

To display all of the Records associated to a particular group, click the dashboard panel title, e.g. Recent Documents link, or the View All... link at the end of the dashboard panel. All of the associated records will be displayed in a Search Results panel on a new tab.

NOTE: If no additional items are available, a warning message No records found will be displayed.

Profile

Users can also access their profile information on the Home Screen. To view the profile information, click the currently logged on user's name and the following information is displayed:

  • Workgroup Server Name - the name of the Workgroup Server the user is connected to.
  • Dataset ID - the dataset ID of the dataset the Content Manager Web Client is connected to.
  • User Type - the user type of the currently logged on user.

Settings

Get Settings

Content Manager Web Client Administrators can set Global Settings that users can download. These Global Settings will overwrite any user settings that have been applied.

The settings that will be modified are:

  • Time Zone
  • Page Size
  • Document Viewer options
  • Dashboard Configuration options

To download the Global Settings:

  1. On the Settings dialogue, click Get Settings.
  2. A warning will be displayed in the Settings dialogue. Click Yes to update the current settings to the Global Settings.
  3. Click OK to close the Settings dialogue.

Dashboard Configuration

The default dashboard panels can be configured via the Settings - Dashboard Configuration dialogue. To configure the displayed dashboard panels:

  1. Click Settings and then click Dashboard Configuration.
  2. Select the dashboard panels to be displayed on the Home Screen dashboard.
  3. Click OK.

NOTE: only 3 dashboard panels can be displayed at any one time. If there are already 3 dashboard panels selected, you will need to clear one option before selecting another.

Locale

Language

Users can change the language Content Manager Web Client is displayed in. To change the language:

  1. Click Settings and then click Locale.
  2. From the drop-down list, select the required Language.
  3. Click OK.

Content Manager Web Client will refresh and reload Content Manager Web Client in the Language selected by the user.

Date Format

Users can specify the format the date is displayed. To select a Date Format, from the drop-down list, select from the following:

mm/dd/yyyy to display the month before the day, e.g. 12/31/2014

dd/mm/yyyy to display the day before the month, e.g. 31/12/2014

yyyy/mm/dd to display the year first, e.g. 2014/12/31

Date Separator

Users can specify the separator used when displaying dates. The separator will be used in the selected Date Format. To select a Date Separator, from the drop-down list, select from the following:

/ to display the date as 31/01/2014

- to display the date as 31-01-2014

. to display the date as 31.01.2014

Time Zone

Users can set their local time zone. Once this option has been set, the displayed dates/times will be in the user's selected time zone.

To set the Time Zone, from the drop-down list, select the appropriate Time Zone. Click OK to save the change.

Page Size

From the drop-down list, users can select the number of items to be displayed on a page of search results. Administrators can set this as a Global Setting, see Global Settings for details.

Document Viewer

User can choose how particular electronic document file types are viewed in Content Manager Web Client. The options to choose from are:

  • Bypass Content Manager Viewer for These File Types - the supported file types will be displayed using the browser's viewing capabilities.
  • Use Onstream Viewer For These File Types - the supported file types will be displayed using the Onstream viewer, and users will be able to utilise the Onstream viewer options, such as Zoom In/Out, Show/Hide Thumbnails etc.

To select the file types:

  1. Click in the option field and from the displayed drop-down list, click the required file type.
  2. Click OK to save the changes.

To remove a selected file type:

  1. In the option field, click the X next to the file type name.
  2. Click OK to save the changes.

Global Settings

Users with appropriate user permissions can set some of the Web Client settings as Global Settings, which users can then download. If a user has the correct user permissions, they will see the Global Settings tab on the Settings dialogue.

The settings that are saved as Global Settings are:

  • Locale tab - Time Zone and Page Size settings
  • Document Viewer tab - Bypass Viewer File Types and Trapeze Viewer File Types options
  • Dashboard Configuration settings

To set Global Settings:

  1. On each of the relevant Settings tabs, apply the settings that are to be made Global.
  2. On the Global Settings tab, select Set as Global Settings.
  3. Click OK.

Masthead menu

The Navigation menu is now the Masthead menu which drops down at the top of the page. It provides links to Content Manager Web Client features, as well as to the Help File. To access a feature, click on the feature name in the Masthead menu.

Alternatively, via a browser, they can be opened by using the following keyboard shortcuts:

  • Home - CTRL + ALT + h
  • Records - CTRL + ALT + r
  • Requests - CTRL + ALT + q
  • Locations - CTRL + ALT + l (L)
  • Saved Searchs - CTRL + ALT + z
  • Check In Styles - CTRL + ALT + y
  • Record Actions - CTRL + ALT + o
  • Workflows- CTRL + ALT + w
  • To Do Items - CTRL + ALT + t
  • User Labels - CTRL + ALT + u

Help file

To access the Content Manager Web Client help file, click the Help button on the Global Search panel or you can open it using the following keyboard shortcut:

  • Help - CTRL + ALT + p
TIP: If accessing the Web Client on a small format screen, the Settings and Help buttons are available from a drop-down menu. Click More to display the Settings and Help options.
