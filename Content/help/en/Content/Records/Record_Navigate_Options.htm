Navigate To Options

When you have performed a search, the Navigate To options allow you to view items that are related or associated with the currently selected record.

To navigate to the associated records:

  1. From the displayed list of records, click the record is to be navigated from.
  1. Click Navigate to and then click the required navigation option.

A new search results window is opened, listing the records that match the selected navigation option.

NOTE: The Navigation options are dependent on the type of record selected as well as the relationships that are associated with the record, so not all options will be available on a record.

Navigation options

  • Related Records - displays the records that are currently related to the selected record
  • Latest Volume - displays the latest volume of the selected record
  • All Volumes - displays all volumes of the selected record
  • All Versions - displays all versions of the selected record
  • Latest Version - displays the latest version of the selected record
  • Current Version - displays the selected record's latest finalized version
  • Linked Documents - displays the records that the selected record's attached electronic document has links to
  • Document Link Of - displays the records that have a document that contains a link to the selected record
  • Container Record - displays the selected record's container record
  • Contained Records - displays the records that are in the selected container record
  • Container Records (+ Alternative) - displays the selected record's container record and alternative containers
  • Contained Records (+ Alternative) - displays the records that are in the selected container record and their alternative containers
  • Records With Same Contacts - displays the records that share the same attached contacts
  • Client records For Contacts - displays the records where the selected Location is the Client Location
  • Client records - for matter records. Displays the client records of the matter record.
  • Matter records - for records that are under matter records. Displays the matter records of the record.
  • Client Matters - only appears when you have selected a client record, it will show all matters for the selected client in a new search.
  • Search contents - for matter records. Displays all records that are contained within the matter record.
  • Records in Email Conversation - displays the records that are attached to the same email communication
  • Series Record - displays the series record of the selected record
  • Records Within Series - displays all records in the selected series record
  • All Redactions - displays all redactions of the selected record
  • Redacted From - displays all original records of the selected redacted record
  • Derivations - displays all documents that were derived from a record that acts as a document template.
  • All Parts - displays all parts of the selected record
  • Latest Parts - displays the latest part of the selected record
