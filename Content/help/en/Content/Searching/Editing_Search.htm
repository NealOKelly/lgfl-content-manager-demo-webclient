Editing Searches

Once a search has been run, you may find that the results list is very large or contains unexpected results so you may want to refine the search clauses to make the search more specific.

Searches can be edited from the:

  • global Search panel, or
  • Search Editor

To edit a search in the global Search panel:

Delete existing search clauses or type in new search clauses and then click Search

The refined results list will be displayed in the relevant Content Manager object type panel.

To edit a search using the Search Editor:

  1. From the global Search panel, click Options and then clickSearch Editor

The <ObjectType> Search dialogue appears.

  1. If a search clause is to be removed, click the delete X option in the search clause pill.
  2. To modify a search clause, in the Build Query field, click the search clause to be modified.
NOTE: if it's a complex search clause, such as a grouped or sub search, you can edit the individual search clauses within the complex clause by selecting each pill, or you can edit the entire complex search clause by selecting the grouped clause.

The Update search clause section appears.

  1. Type in the changes to the search criteria, or select/clear any available options, once the changes to the search clause are complete, click Finish updating search part.

The selected search clause will be updated in the Build Query field.

  1. Further refine your search by adding additional search clauses, see Search Editor for details; or using Advanced Search options to combine the search clauses; or Filter/Sort the output.
  2. Click Search to run your refined search.

The refined results list will be displayed in the relevant Content Manager object type panel.

NOTE: all search clauses can be cleared from the Build Query field by clicking Clear.
