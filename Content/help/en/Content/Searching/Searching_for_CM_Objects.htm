Searching for Content Manager Objects

To perform a search with Content Manager Web Client:

  1. From the global Search panel, select the Content Manager object to search for:
    • Record
    • Location
    • Activity
    • Check In Style
    • Consignment Approver
    • Consignment Record Rejection
    • Historical Event
    • Record Action
    • Request
    • Saved Search
    • To Do Item
    • User Label
    • Workflow
  1. In the Search query field, type the search method and parameter, for example CreatedOn:Today or title:Reef
  2. Click Search to run the search, or press Enter on the keyboard.

The search results are displayed in the Search Results panel.

NOTE: When a search returns more than 500 results, a Count option will become available next to the "Showing Results..." caption and above the search results list. To count the total number of results returned in a search, click Count and the results will be displayed as [Total Results:xxx]. This Count option is available for all Content Manager object pages with the exception the Workflow page.

Sample Search Methods for Records

Title word - search record(s) for any word in a record title, e.g. title:admin*

Any word -search for record(s) combining title words and notes words, e.g. anyword:nest

Assignee - search record(s) based on the Assignee value, e.g. assignee:abbott*

Record Number - search for record(s) using it's record number e.g. number:FBOX15/1

Date Registered - search for record(s) using the date registered value, e.g. registeredOn:this week

Date Created - search for records using the date created value, e.g. createdOn:06/01/2015 to 06/09/2015

Sample Search Methods for Locations

Surname - search for Location(s) based on the Last name, e.g. surname:abb*

Email Address - search for Location(s) based on their email address, e.g. email:petera@localhost.com

First Name - search for Location(s) based on their first name, e.g. givenNames:rhoda

Sample Search Methods for Activities

Assigned To - search for Activities based on the Location the Activity is assigned to, e.g. assignee:me

Due Date - search for Activities based on their due date, e.g. dueOn:today

Sample Search Methods for Check In Styles

Check In Style Owner - search for Check In Style(s) based on the owner, e.g. owner:Abbey, Rhoda (Ms)

Linked Record - search for Check In Style(s) based on the Container record, e.g. container:15/8901

Sample Search Methods for Historical Events

For record - search for Historical Events on a particular record, e.g. record:G16/1

Updated by - search for Historical Events made by a particular user, e.g. user:"abbott, peter"

Sample Search Methods for Record Actions

Action Name - search for a Record Action based on the Record Action name, e.g. name:Accident Report

Done - search for all completed Record Actions, e.g. done

Ready to Start - search for all Record Actions that are ready to start, e.g. canstart

Completed on - search for Record Actions completed on a particular date or within a date period, e.g. doneOn:this week

Sample Search Methods for Requests

Record Assignee - search for Request(s) based on the Record's Assignee, e.g. assignee:me

Date Required - search for Request(s) based on the date the Request is required, e.g. dueOn:this week

Status - search for Request(s) based on the Request status, e.g. status:"New Request - Awaiting Approval", "New Request - Not Yet Actioned"

Priority - search for Request(s) based on the Request priority, e.g. priority:high

Sample Search Methods for Saved Searches

Name - search for Saved Searches by name. Use a wildcard stand in for one or more unspecified characters.

Favourite - search for your favourite saved searches, e.g. favourite

Object type - search for saved searches that are for a specific Content Manager object type, e.g. type:location

Sample Search Methods for To Do Items

Date Due - search for To Do Item(s) based on the date the item is due, e.g. dueOn:tomorrow

To Do Item name - search for To Do Item(s) based on the To Be Done field, e.g. name:help*

Assignee - search for To Do Item(s) based on the assignee e.g. assignee:me

Sample Search Methods for User Labels

Owner - search for all User Labels that are owned by a particular user, e.g. owner:"abbott, peter"

User Label Name - search for User Labels by their name, e.g. name:project*

Sample Search Methods for Workflows

Due Date - search for Workflow(s) based on their due date, e.g. dueOn:this week

Overdue - search for Workflow(s) that are Overdue, e.g. overdue

Initiating record - search for Workflow(s) based on an initiating record, e.g. initiator:15/365

Date Completed - search for Workflow(s) based on the date the Workflow was completed, e.g. doneOn:last month
